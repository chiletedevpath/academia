# Otros

Espacio reservado para proyectos académicos o formativos que no pertenezcan directamente a UTP o Tecsup.

## Uso esperado

Registrar aquí solo contenido con contexto claro:

- Cursos externos.
- Talleres independientes.
- Certificaciones.
- Retos académicos especiales.

## Mantenimiento

Antes de agregar un proyecto en esta sección, confirmar si realmente no pertenece a una institución o curso ya existente.
