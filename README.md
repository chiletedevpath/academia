# Academia

**Academia** reúne los proyectos formales que desarrollé como parte de mi formación en UTP, Tecsup y otros espacios de estudio.

Este repositorio ayuda a entender qué construí, en qué curso nació cada proyecto y qué aprendí en el proceso. No busca llenar GitHub de carpetas: busca conservar evidencia técnica útil, ordenada y fácil de revisar.

## Cómo leer este repositorio

- `utp/` conserva proyectos desarrollados durante la carrera de Ingeniería de Sistemas e Informática.
- `tecsup/` conserva entregas y prácticas formales de cursos técnicos.
- Cada proyecto importante tiene su propio README con contexto, tecnologías, ejecución y aprendizaje aplicado.

## Relación con Chilete DevPath
